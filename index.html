Setup Guide — Deploy in ~5 minutes
1
Get a Supadata API key
Go to supadata.ai and create a free account. Free tier includes 100 transcript requests — plenty for personal use. Copy your API key from the dashboard.
2
Create a new Cloudflare Worker
Go to dash.cloudflare.com → Workers & Pages → Create → Create Worker. Give it a name like lian-dao-hao-transcript. Click Deploy.
3
Paste the Worker code
Click Edit code. Delete everything in the editor and paste the contents of worker.js from your repo. Click Deploy.
4
Add your Supadata API key
In your Worker's settings, go to Settings → Variables → Add variable. Name: SUPADATA_API_KEY, Value: your Supadata key. Click Save and deploy.
5
Paste your Worker URL here
Your Worker URL is shown at the top of the Worker dashboard — looks like https://lian-dao-hao-transcript.yourname.workers.dev. Paste it in the field above and click Test.
